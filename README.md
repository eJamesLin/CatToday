# CatToday (Adogcat)
#### HackNTU2015參賽作品
1. 運用 iOS Today Extension，讓可愛待領養狗狗貓貓出現在iPhone通知中心:p
2. 多種經典小語，英語格言，一句話惹惱工程師，讓通知中心更有趣！
3. 看到喜歡的寵物，點擊即可觀看領養資訊～
